# Smart Exam - Online Examination System

## 📌 Project Description
This is a web-based Online Examination System developed using PHP and MySQL.  
It allows admin, teachers, and students to manage and participate in online exams.

## 👨‍💻 Modules
- Admin Panel
- Teacher Panel
- Student Panel

## ⚙️ Technologies Used
- PHP
- MySQL
- HTML, CSS, JavaScript

## 📂 Project Files
All project files are available in the uploaded ZIP file.

## 🎯 Purpose
This project is developed as a Final Year Project for BCA.
